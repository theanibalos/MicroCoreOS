## 🏗️ Philosophy and Architecture
- **Resilient Core**: The Kernel and Container are the stable heart.
- **Lifecycle**: `setup` (Tools) -> `__init__` (Plugins) -> `on_boot` (Registration) -> `execute` (Action).
- **Dependency Injection**: Plugins receive tools in the constructor. **Check signatures below.**

## 🚀 How to create a new Domain & Plugin
To extend the system, follow this pattern:

1. **Create Directory**: `domains/[domain_name]/plugins/`
2. **Plugin Lifecycle Pattern**:
```python
from core.base_plugin import BasePlugin

class MyNewPlugin(BasePlugin):
    def __init__(self, logger, event_bus, http, db): # 1. DI Phase
        self.logger = logger
        self.bus = event_bus
        self.http = http
        self.db = db

    def on_boot(self): # 2. Registration Phase
        self.http.add_endpoint('/my/path', 'GET', self.handler)
        self.bus.subscribe('some.event', self.execute)

    def execute(self, data=None): # 3. Action Phase
        self.logger.info('Performing work...')
        return {'success': True}

    def handler(self, data, context):
        return self.execute(data)
```
